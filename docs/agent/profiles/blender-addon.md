# Blender Add-on Profile

Profile id: `blender-addon`.
Status: dormant.

This profile prepares future planning for a Blender add-on. It does not activate
Blender, Python packaging, background tests, add-on registration code, or zip
export commands.

When a future request activates this profile, decide:

- supported Blender versions;
- Python package layout;
- `bl_info` ownership;
- registration and unregistration contract;
- background test command;
- packaging and release zip format;
- add-on preferences and asset paths;
- artifact handling in `DO_NOT_PUSH.md`.
